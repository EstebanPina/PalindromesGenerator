# PalindromesGenerator
Generador de palindromos binarios de longitud n&lt;100000 desarrollado en python.
Reglas de produccion:
(1) P -> e
(2) P -> 0
(3) P -> 1
(4) P -> 0P0
(5) P -> 1P1.
